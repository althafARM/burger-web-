# TFC Nation - Tender Fried Chicken Website

## Project Description
TFC Nation is a simple restaurant website for Tender Fried Chicken.  
The website displays information about the restaurant, products, services, and contact details.

## Features
- Responsive navigation bar
- Hero banner section
- Features section
- About section
- Product menu
- Services section
- Contact form
- Social media footer

## Technologies Used
- HTML5
- CSS3
- SVG Icons
